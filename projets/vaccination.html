projet Vaccination

Le lycée Bertran de Born est centre de vaccination pour les jeunes scolaires. L’application va maintenant gérer une base de données dont le script est fourni par le fichier « base_vaccination.sql ».

Nous avons dans un premier temps implémenté une base de données, puis mis en place les fonctionnalités décrites dans la suite du descriptif.
